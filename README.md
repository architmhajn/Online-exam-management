# Online Examination System

A secure exam platform with anti-cheating for colleges.

## Features
- Roles: Admin (assign roles), Instructor (create exams), Student (take exams).
- Anti-cheating: Tab-switch, fullscreen, webcam, timers.
- Automatic/manual grading.

## Setup
1. Install Node.js, MySQL.
2. Run SQL scripts to create DB.
3. `cd backend && npm install && npm start`
4. `cd frontend && npm install && npm start`
